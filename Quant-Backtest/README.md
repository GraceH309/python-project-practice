**Project Introduction**

A quantitative backtesting framework, including market data downloading, technical indicator calculation, trading strategy development, backtest engine, risk indicator calculation and result visualization.



**Features**

1.Automatic download and data cleaning of stock quotes

2.Calculation of moving average, return rate and other technical indicators

3.Double moving average trading strategy implementation

4.Simulated trading, transaction fee deduction and position management

5.Performance metrics calculation: annualized return, Sharpe ratio, maximum drawdown and other risk indicators

Environment \& Dependencies



**How to Run**

Install dependencies:

bash

pip install yfinance pandas numpy matplotlib scipy

python main.py
