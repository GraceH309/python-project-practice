\# Algorithm + Quant + Trading System Demo\## Project OverviewThis repository contains three independent practical projects, covering algorithm visualization, financial quantitative backtesting and backend trading systems.1.Algorithm-Visualizer: Animated visualization of Bubble Sort and Dijkstra's Shortest Path algorithm2.Quant-Backtest: A full quantitative strategy backtesting framework, including technical indicators, trading strategies, risk analysis and data visualization3.Trading-System: A lightweight trading backend built with FastAPI, featuring data pipelines and RESTful APIs\## Unified Environment DependenciesOpen CMD or terminal on your device, and run the following command to install all required packages:```bashpip install matplotlib numpy yfinance pandas scipy fastapi uvicorn
